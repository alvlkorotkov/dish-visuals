# Dish Visuals

Visual catalog for cinematic vegetarian dishes: plated concepts, serving angles, and recipe notes for a closed-dinner moodboard.

Published preview: https://alvlkorotkov.github.io/dish-visuals/

## Structure

- `index.html` - published landing page.
- `dishes.html` - same dish catalog as a direct page.
- `assets/dishes/` - dish image folders.
- `robots.txt` - asks crawlers not to index the preview.

## Image Set

The target set for each dish is five angles:

- `top-shot.png`
- `close-up.png`
- `table-scene.png`
- `chef-hands.png`
- `eating-scene.png`

Some dishes are still in progress and currently have only the first angle.
